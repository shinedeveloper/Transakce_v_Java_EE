Priklady k prednace Transakce v Java EE z brnenskeho CZJUG (2.4.2013)

Pokud si to budete chtit spustit, potrebujete:
 1. Rozbalit soubor Java-Training.7z do C:\Java-Training
 2. Vsechny priklady z repository by mely byt checkout-ovany do C:\Java-Training\Projects

V C:\Java-Training\MySQL je MySQL
  - spustite ji pomoci C:\Java-Training\MySQL\startup.bat
V C:\Java-Training\glassfish.v3 je GlassFish
  - spustite take pomoci C:\Java-Training\glassfish.v3\startup.bat
V C:\Java-Training\IntelliJ-IDEA je editor IntelliJ IDEA.
  - spustite pomoci C:\Java-Training\IntelliJ-IDEA\startup.bat
